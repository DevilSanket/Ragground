## Ingredients

- Ground Up Cheat Code (miso base)
- Noodles
- Hot water

## Instructions

1. Prepare the Ground Up Cheat Code base in a bowl.
2. Cook noodles according to package instructions.
3. Combine cooked noodles with the prepared Ground Up Cheat Code base and hot water to create a soup.
4. Add optional veggies and tofu.
5. (Optional) Stir in a splash of rice vinegar for added balance.
6. Mix thoroughly before enjoying to ensure all flavors are combined.
